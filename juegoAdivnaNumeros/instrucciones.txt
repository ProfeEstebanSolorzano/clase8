Instrucciones del ejercicio

Crear un juego con html y JS en el cual se pida al usuario que ingrese un número. 
Este número se compara con un número elegido al azar por el programa. 
Si el número es mayor, se le dice al usuario y se resta una oportunidad
Si el número es menor, se le dice al usuario y se resta una oportunidad
Si el número es igual, entonces se gana el juego
Si se le acaban las oportunidades, pierde el juego

Para empezar, hay que darle click a un botón
Se crea un input tipo text donde se escribe el número a adivinar
Este input va a estar "disabled" hasta que se inicie el juego
Una vez que el juego termine, sea que gane o pierda, el botón vuelve a estar "disabled"

Para el juego tenderemos un solo botón al que le va a cambiar el texto y la función dependiendo del momento del juego
